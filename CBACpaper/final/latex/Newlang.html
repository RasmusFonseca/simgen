How do I adapt tex2doc for a different language?

In order to make tex2doc deal properly with your language, you have to edit the macro tex2doc. (1) Search for "~~~", in order to find the appropriate locations in the source code. (2) Insert the proper values. (To find out the search patters, open the find dialog, and select paragraph, non-breaking space, and tabulator from the "special" menu.)

The construct, which has to be edited, looks like this:

  if [check for language1] then
    [definitions1]
  elseif [check for language2] then
    [definitions2]
  elseif [check for your language] then
    [your definitions]
  else
    [alert]
  endif

Thus, as an example, the following two lines:

  ElseIf xlanguage$ = "~~~" Then
     xr$ = "~~~"

should become - in case you write in French -

  ElseIf xlanguage$ = "Français" then
    xr$ = "^p"
           ^^ or whatever is the correct pattern.

In case you don't use inputenc or something similar, you might also want to edit the procedure "languageSpecificChars".
